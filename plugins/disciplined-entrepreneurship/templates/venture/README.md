# Venture Workspace — `{{VENTURE_NAME}}`

This folder is the single source of truth for validating one startup idea using the
**Disciplined Entrepreneurship (DE)** framework. It is a living workbook: update it as
you learn, not once at the end. Evidence beats opinion — every claim here should trace
back to a customer interview (`interviews/`), a data source (`research/`), or a labelled
assumption (`assumptions.md`).

Supporting evidence lives in folders:

- **`research/`** — secondary research (market reports, stats, filings). One note per source, with the URL and date.
- **`interviews/`** — primary market research. One note per customer conversation. This is the most important folder — DE is built on talking to real potential customers.
- **`competitors/`** — one profile per competitor, plus the comparison matrix.

## How to work this workspace

1. **Go in order, but stay loose.** The 24 steps build on each other (you can't size a
   market before you've defined the customer). But validation is iterative — expect to
   loop back and revise earlier steps as interviews teach you things.
2. **Pick ONE beachhead market** (Step 2) and resist the urge to keep options open.
   Dominating a tiny, homogeneous market beats a sliver of a huge one.
3. **Talk to customers before you build.** Inquiry, not advocacy. Don't pitch — learn.
4. **Quantify.** TAM, value proposition, LTV, and COCA are numbers, with stated
   assumptions and sources, not adjectives.
5. **Update the dashboard** after each work session so progress and key metrics stay visible.
