# Research — Secondary Market Research

Desk research that supports (but never replaces) primary research. Use it to size markets,
understand industry structure, and find people to interview.

## Capture provenance — always

One note per source. Name files descriptively (e.g. `gartner-2025-market-size.md`).
Every quantitative claim that ends up in the venture docs must trace back to a note here.

```markdown
# {{Source title}}
URL: {{link}} · Publisher: {{who}} · Date accessed: {{YYYY-MM-DD}} · Published: {{date}}
Reliability: {{primary data / vendor estimate / blog / forum — judge it}}

## Key facts (with numbers)
- …

## How it's used in this venture
- e.g. feeds Step 4 TAM bottom-up estimate (# end users), Step 16 pricing benchmark, etc.
```

## Where to look

- **Market size & industry:** government statistics, trade associations, census/economic
  data, public company filings (10-K/S-1), analyst summaries.
- **Competitors & funding:** Crunchbase, PitchBook, company sites, job postings.
- **Customer voice at scale:** G2 / Capterra / Trustpilot reviews, app-store reviews,
  Reddit and niche forums, support communities.
- **Demand signals:** Google Trends, keyword search volume, ad costs.

Triangulate. When data is sparse, build a **bottom-up** estimate from defensible unit
assumptions and clearly label every assumption. See the `market-research` skill.
