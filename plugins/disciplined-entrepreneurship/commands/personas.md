Use the **disciplined-entrepreneurship** and **market-research** skills to build the **End User
Profile (Step 3)** and **Persona (Step 5)** for venture **$ARGUMENTS**.

1. Read `references/theme-1-who-is-your-customer.md` for the profile/persona method and fields.
2. If the venture's `interviews/` folder has notes, dispatch the **`customer-discovery-analyst`** agent
   to synthesize them into an evidence-based profile and a single named persona with **ranked
   purchasing criteria**. If there are too few interviews, generate a tailored interview guide instead
   and state plainly that real conversations are needed first (a persona without PMR is fiction).
3. Write the profile and persona into `01-who-is-your-customer.md`, ensuring the ranked purchasing
   criteria are explicit (they drive Steps 6, 7, 8, and 11), and update `dashboard.md`.
