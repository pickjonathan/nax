The discipline checkpoint before committing to the plan. Use the **market-research** skill and the
**`customer-discovery-analyst`** agent for venture **$ARGUMENTS**.

1. **Talk to ~10 real prospects.** If interviews are thin, generate a tailored interview guide and
   help line up conversations (past behaviour and specifics, never pitches). Log each in
   `interviews/`.
2. **Validate (Step 9, "Next 10 Customers").** Synthesize: is this a **real, homogeneous market**
   with genuine demand ("love, not just like")? Be intellectually honest — don't pretend a
   heterogeneous market is homogeneous.
3. **Loop back if it doesn't hold.** Update the End User Profile / Persona (Steps 3, 5) or return to
   **`/de-specify`** if reality differs from the assumption.
4. Record validated/invalidated hypotheses in `assumptions.md`; mark **Step 9**. Once ~10 interviews
   validate the beachhead, the clarify gate passes. Finish by showing `/de-next <slug>`.
