Use the **disciplined-entrepreneurship** skill (`references/theme-5-how-you-build.md` and
`references/theme-6-how-you-scale.md`) for venture **$ARGUMENTS**. Proceed after de-risking — if you
haven't run `/de-analyze`, do that first (check `/de-next <slug>`).

- **Step 22 — MVBP.** Define the *minimum viable business product*: the smallest thing that
  **delivers value in use**, **captures value (the customer pays)**, and **starts a feedback loop**.
  Core features only — resist scope creep.
- **Step 23 — "the dogs eat the dog food."** Instrument real usage, payment, retention, and
  word-of-mouth — behavioral evidence, not survey intent. Record in `05-how-you-build.md`.
- **Step 24 — product plan.** Write the prioritized roadmap + follow-on-market sequence in
  `06-how-you-scale.md`; don't abandon the beachhead early.

Mark steps **22–24**; update `00-summary.md` and `dashboard.md`. When raising, run `/pitch-deck
<slug>`. Finish by showing `/de-next <slug>`.
